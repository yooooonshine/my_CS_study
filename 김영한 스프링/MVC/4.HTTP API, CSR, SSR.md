# HTTP API
html이 아니라 데이터를 전달할 때 사용되며, json 형식으로 전달된다.

![[Pasted image 20240219001826.png]]
http api는 다양한 시스템에서 호출될 때 사용된다. 즉 웹 페이지나, 앱 페이지는 서버가 따로 존재하고, was에는 단순 데이터만 전달 요청할 때 http api가 사용될 수 있다.

# 서버사이드 렌더링(SSR)
이는 서버가 DB에서 정보를 조회해, 데이터를 이용하여 html을 만들어 클라이언트에게 제공하는 방식이다.

* 주로 정적인 화면에서 사용한다.
* 관련된 기술은 타임리프, jsp등이 있다.
# 클라이언트 사이드 렌더링(CSR)
html 결과를 자바스크립트를 사용해 웹 브라우저에서 동적으로 생성하여 적용하는 방식. 주로 동적 화면에서 사용한다.
* 관련 기술은 리엑트 등이 있다.
